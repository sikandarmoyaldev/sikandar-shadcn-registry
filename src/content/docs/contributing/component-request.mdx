---
title: Component Request
description: Request a component you think should be added.
---

import { Steps } from "@astrojs/starlight/components";
import { PUBLIC_GITHUB_URL } from "astro:env/client";

<Steps>

1. **Check Existing Components**

    Review the [components list](/components) and the <a href={`${PUBLIC_GITHUB_URL}/discussions/categories/component-request`}>GitHub discussions tab</a> to see if the component is already implemented or requested.

2. **Open a Discussion**

    Go to the GitHub repository and <a href={`${PUBLIC_GITHUB_URL}/discussions/new?category=component-request`}>start a new discussion</a>.

3. **Describe the Component**

    Provide a clear description, use cases, and any references or examples. The more detailed a request is, the easier it is for maintainers to understand and implement.

</Steps>
